Uživatel se nacházi v Registračním formuláři

Doplň validní data pro registraci. Pozn: je možné, že uživatel už existuje. V datech je pouze ukázkový příklad.

username "testUser"

email "test@test.com"

password "Heslo1234"

confirm-password "Heslo1234"

 

Odeslání dat a zobrazení okna: “Uživatel vytvořen.”

Zkontroluj v DB vytvoření nového klienta.

Databaze: DatabaseUser.db, SELECT * FROM Users WHERE username = ‘testUser’ –-Doplň vytvořeného uživatele

-Klient se nachází v databázi a data odpovídají datum na vstupu. - Password a Password again jsou zapsané jako hash – Password a Password again jsou totožné
